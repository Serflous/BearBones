Detailed Description

The singleton class for managing the input for the engine. Stores the state of each mouse and keyboard to be polled at will.

Input::KeyState Input::InputManager::GetKeyState ( int  key)

Gets the current key state of the key.

Parameters
[in]keyThe key to check.
Returns
The state of the key. Will be KS_KEY_PRESSED on first check, then it updates to KS_KEY_REPEAT for every subsequent check.
Note
The key is an integer. Can pass a char in place and it will use the ASCII value automatically.
